== Интернет-магазин книг на Ruby on Rails

Этот интернет-магазин создан на основе материал книги C. Руби, Д. Томас, Д. Хэнссон "Гибкая разработка веб-приложений в среде Rails"

== Установка

* Склонируйте проект и перейдите в его папку

    $ git clone git@github.com:yesnik/depot.git
    $ cd depot

* Установите гемы для проекта

    $ bundle install

* Создайте БД на основе схемы и импортируйте тестовые данные

    $ rake db:setup

* Запустите сервер разработки

    $ rails s
